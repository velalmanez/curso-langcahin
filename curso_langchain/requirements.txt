1. Creacion de entorno virtual 
   python -m venv env

2. Activacion de entorno virtual
    - En Windows:
      .\env\Scripts\activate
    - En macOS/Linux:
      source env/bin/activate

3. Instalacion de dependencias
    - pip install langchain==0.3.* langchain-google-genai streamlit
    - pip install pyPDF2
